# Reproducibility Statement

This repository accompanies the paper:

"An Explainable Multi-Agent Multimodal AI Framework for Trustworthy Assistive Urban Navigation of Visually Impaired Users in Smart Cities."

The repository provides resources supporting transparency, reproducibility, and independent verification of the reported findings.

Included materials:

- Framework architecture
- Multi-agent workflow pseudocode
- CARLA simulation configuration
- Experimental protocol
- Evaluation metrics
- Ablation study methodology
- Statistical validation procedures
- Result summaries and visualisations

All experiments were conducted using the BlindWays dataset within the CARLA Urban Driving Simulator.

Researchers may use these resources to reproduce the methodology and compare future navigation systems against the proposed framework.
