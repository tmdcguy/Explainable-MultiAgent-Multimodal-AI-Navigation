# Explainable-MultiAgent-Multimodal-AI-Navigation
Reproducibility resources for an Explainable Multi-Agent Multimodal AI Framework for Trustworthy Assistive Urban Navigation of Visually Impaired Users.


## Overview

This repository accompanies the paper:

**"An Explainable Multi-Agent Multimodal AI Framework for Trustworthy Assistive Urban Navigation of Visually Impaired Users in Smart Cities."**

The repository provides resources supporting transparency, reproducibility, and independent verification of the proposed framework.

---

## Dataset

- BlindWays

## Simulation Environment

- CARLA Urban Driving Simulator

---

## Main Results

| Metric | Value |
|----------|---------|
| Navigation Accuracy | 96.8% |
| Navigation Success Rate | 95.6% |
| Route Completion | 97.2% |
| Collision Avoidance | 98.1% |
