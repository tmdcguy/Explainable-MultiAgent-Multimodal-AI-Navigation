# Evaluation Protocol

Dataset:
- BlindWays

Simulator:
- CARLA Urban Driving Simulator

Evaluation Metrics:
- Navigation Accuracy
- Navigation Success Rate
- Route Completion
- Collision Avoidance
- Explainability Metrics
- Computational Performance

Statistical Tests:
- Independent t-test
- One-way ANOVA
- Wilcoxon Signed-Rank Test

Significance Level:
p < 0.05
